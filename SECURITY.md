# Security Policy

## Reporting a vulnerability

If you believe you have found a security vulnerability in this project, please report it privately. **Do not open a public GitHub issue for security-sensitive matters.**

### How to report

- **Email:** manasabehera5901@gmail.com  
- **Subject:** Prefer including something like `[SECURITY] Structural Gate` so the report can be triaged quickly.

Please include:

- A short description of the issue.
- Steps to reproduce (if applicable).
- Impact (e.g. what an attacker could do).
- Any suggested fix or mitigation, if you have one.

### What to expect

- We will acknowledge your report and work with you to understand and address the issue.
- We may ask for clarification or more details.
- We will not disclose your report publicly before a fix is available, unless you agree otherwise.

### Important

**Security issues must not be filed as public issues.** Use the contact above only. Thank you for helping keep this project and its users safe.
